# CS-300
The objective of the two projects for this course were to create an application which would utilize different data types for organizing and displaying data. In the case of these projects, a Binary Search Tree (BST) was used to organize, sort and display a collection of computer science courses offered by ABCU University. After analyzing the options for different data structures within this course the BST was chosen for its efficiency given the specifications of the projects. The hash table and vector data structure options were not as optimal for this specific project. Overcoming roadblocks involved carefully checking over pieces of code that may have had logic errors present. To ensure that the application would operate as expected, temporary print lines were used to give outputs during the applications execution. These were removed when functionality met the requirements for the project. This course and both projects strengthened my skills in programming while introducing new skills in data managment types. These projects allowed me to learn about different data structures and what data structure is most optimal based on the project requirements. These projects help reinforce my skills in keeping code properly organized through effective use of blank space and in-line comments to describe aspects of the code to ensure that a user who has never seen the code before could understand and modify the application if needed.
